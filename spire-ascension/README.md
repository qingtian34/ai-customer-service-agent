# 尖塔征途 (Ascension Spire)

受《杀戮尖塔》启发的**商业级体量** Roguelike 卡牌构筑游戏，纯前端实现。

## 内容规模

| 类别 | 数量 | 说明 |
|------|------|------|
| 英雄 | 4 | 铁卫 / 影刺 / 奥法 / 血猎 |
| 卡牌 | 80+ | 含基础、专属、无色、诅咒/状态牌 |
| 遗物 | 45+ | 普通 / 罕见 / 稀有 / Boss |
| 敌人 | 25+ | 普通、精英、三幕 Boss |
| 事件 | 18+ | 多选项风险收益抉择 |
| 药水 | 13 | 战斗中可使用 |
| 进阶 | 0–5 | 逐级提升难度 |

## 核心系统

- **分支地图**：战斗 / 精英 / 商店 / 休息 / 事件 / 宝箱 / Boss
- **回合制战斗**：能量、格挡、意图、状态（力量/易伤/中毒/无形等）
- **能力牌**：持续整场战斗的被动（腐化、恶魔形态、电动力学等）
- **Boss 遗物**：每幕 Boss 后三选一
- **进阶模式**：更高精英率、更强敌人、更少资源
- **运行统计**：通关/败北记录、战斗数据、最高进阶

## 运行

```bash
cd spire-ascension
python -m http.server 8765
```

打开 http://localhost:8765 ，或双击 `start.bat`。

## 操作

1. 选英雄 → 选进阶等级 → 地图选路
2. 战斗：点敌人选目标 → 点手牌打出 → **E** 结束回合
3. 点击底部**药水**图标在战斗中使用
4. 战后选牌 / 精英掉遗物 / Boss 后选 Boss 遗物
